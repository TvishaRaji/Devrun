Library Management System

The Library Management System is a web application built with Angular, Flask, JSON Server, and SQLite. It provides a comprehensive set of features for managing library resources, including books, authors, categories, and user accounts. The system integrates with the Google Books API for book search functionality and offers data visualization using Power BI.

Pages

  • Home: The landing page of the application.
  • Landing: A page for searching books using the Google Books API.
  • Google API Books: Displays search results from the Google Books API.
  • Category: Allows browsing and filtering books by category.
  • Product: Shows detailed information about a specific book.
  • Profile: Displays user profile and favorite books.
  • Login and Signup: Pages for user authentication and registration.
  • Docs: Documentation and guides for using the application.

User Flow

  1. Sign up and log in to the website.
  2. Navigate to the Docs page to access documentation and guides.
  3. Go to the Category tab to browse books by category.
  4. Utilize the search bar to find specific books and sort the results.
  5. Click on a book to view its details on the Product page.
  6. Click on additional links to navigate to related sections or pages.
  7. Find the paginator in the footer for easy navigation through book listings.
  8. Explore the footer for a link to the Power BI app for data visualization.
